# Fastcoding VN Frontend Test

Real estate landing page from Figma, implemented with plain HTML, CSS and vanilla JavaScript.

## Requirements covered

- Plain HTML/CSS, no CSS framework and no frontend framework.
- Vanilla JavaScript only for mobile menu, scroll active menu, and reveal animation.
- Responsive layout for PC, tablet and smartphone.
- Relative paths for source files: `./css/style.css`, `./js/main.js`.
- Clean folder structure.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy

Upload this folder to Netlify, Vercel static hosting, GitHub Pages, or any static web server.
